FrontEnd Task Setup
==================

1. Clone the Repository
-----------------------
git clone https://github.com/sazid62/frontEnd-Task.git

cd frontEnd-Task

2. Install Dependencies
-----------------------
npm install

3. Start the Application
------------------------
npm run start

Important Note
--------------
⚠️ Make sure to download and start the backend **before** running the frontend.

Backend repository: https://github.com/sazid62/task-backend
